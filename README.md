# Market-X Frontend

## Description
Market-X Frontend is a responsive web application built with React.js. It provides a modern user interface for browsing products and interacting with the backend API.

## Features
- Responsive UI
- Product listing and details
- Authentication flow (login/register)
- Protected routes

## Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Tailwind CSS
- React Router

## Backend Integration
 - Connects to the Market-X backend via RESTful APIs
 - Handles authentication, product management, and data fetching
